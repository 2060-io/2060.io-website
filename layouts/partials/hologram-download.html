Some Hologram features

  • Create chat connections with people, credential issuers and conversational services.
  • Collect verifiable credentials from issuers and store then securely in your wallet.
  • Present verifiable credentials, send text, voice messages, photos, videos and files to your connections.

By combining Verifiable Credentials and messaging, users can create fully authenticated chat connections where both parties are clearly identified.

Hologram is free software and is part of the 2060.io project.

Developers can reach our Github repository to know more about the 2060.io project and learn how to create their own DIDComm based trustable conversational services.
